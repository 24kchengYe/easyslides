# Style Packs

Style packs are reusable visual systems, not page layout packs. Keep them out
of `templates/layouts/` so they do not appear in the PPT Master page-template
index.

- `l001_notebook_defense/`: wine-red notebook-defense SVG shells and tokens.
- `guizang_ppt/`: Guizang Style A and Style B editable-PPT migration, including
  the Swiss `S01`-`S22` layout registry.
